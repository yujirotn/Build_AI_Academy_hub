## チェックリスト

### 台本との整合性チェック（最重要）
- [x] 台本の内容がすべてスライドに反映されている
- [x] 台本にない内容を追加していない
- [x] スライドの順序が台本の順序と完全に一致
- [x] 各スライドのCOPYが台本の文言に忠実

### 基本チェック
- [x] すべてのスライドにタイトルがある
- [x] すべてのスライドにVISUAL要素が指定されている
- [x] 1スライドあたり台本約200〜250文字に対応

### 3ブロック分離チェック
- [x] COPYに「レイアウト」「図解」「アイコン」「配置」等の指示語が含まれていない
- [x] VISUALに「〜です」「〜ます」等の文章表現が含まれていない
- [x] LAYOUTに表示テキストが含まれていない

### まとめチェック
- [x] まとめスライドで要点が台本どおりに集約されている
- [x] クロージングの挨拶が含まれている
