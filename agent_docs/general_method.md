# 概念系台本の書き方

## 概要

概念系台本は、AIの基礎知識や考え方、 法律関連、業界動向などを解説する台本です。

**例**: 生成AIとは何か、プロンプトの基本、生成AIと著作権

---

## 台本フォーマット（必須）

すべての台本は以下の形式で作成すること。

### 基本構造

```markdown
# [台本タイトル]

ジャンル: [概念系 / ツール解説系 / ユースケース系 / 比較系]
想定時間: [XX分 または XX〜XX分]
作成日: [YYYY年M月D日]

## 1. オープニング

【スタート】こんにちは、Build AI Academyのたんのです。

[本文...]

## 2. [セクション名]

[本文...]

### [サブセクション名]（必要に応じて）

[本文...]

## N. クロージング

[まとめの本文...]

【終わり】それでは次の講座で会いましょう。ありがとうございました。
```

### フォーマットルール

| 項目 | ルール |
|------|--------|
| タイトル | `# [台本タイトル]`（H1） |
| 冒頭情報 | ジャンル・想定時間・作成日のみ。箇条書きにしない |
| セクション | `## N. セクション名`（1から連番） |
| サブセクション | `### サブセクション名` |
| 区切り線 | **使用しない**（`---` は入れない） |
| 【スタート】 | 1. オープニングの冒頭、最初の発話の前に配置 |
| 【終わり】 | 最終セクションの最終文の冒頭に配置 |
| 画面指示 | `（画面：〜）` 形式。ツール解説系で使用 |

---

## 台本構成のポイント

### 1. 前提知識を先に説明する

結論やリスクを説明する前に、その話を理解するために必要な「前提となる仕組み・背景」を先に説明する。

**例**: 
- ×「AIは学習データをうっかり書き写すリスクがある」→ なぜそうなるかが分からない
- ○「AIは膨大なデータを学習してパターンを覚えている」→「だから学習データに似たものが出る可能性がある」

**原則**: 「なぜそうなるのか」を先に説明してから、「だからこうなる」という結論を述べる

### 2. 専門用語・堅い表現を避ける

法律・技術など専門性の高いテーマでも、視聴者に「難しそう」「自分には関係なさそう」と思わせない表現を心がける。

**避けるべき表現**:
- 法律用語（判例、依拠性、類似性など）→ 初出時は必ず平易な言葉で説明を添える
- 具体的な事件名・判例名 → 抽象的な事例として紹介する
- 「〇〇法第△条」のような条文引用 → 原則として使わない

**推奨される表現**:
- 例え話を活用する（「魔法の筆」「スポーツカーと交通ルール」など）
- 「つまり〇〇ということです」と言い換えを入れる
- 専門用語には必ず読み仮名と簡単な説明を添える

### 3. 事例は「両面」を挙げる

ある判断基準を説明するとき、一方向の事例だけでなく、反対の事例も挙げることで理解が深まる。

**例**:
- 著作権侵害の話 → 「セーフの事例」と「アウトの事例」の両方を紹介
- ツールの使い方 → 「うまくいく例」と「失敗する例」の両方を紹介

**効果**: 視聴者が「この場合はどっちだろう？」と自分で考えられるようになる

### 4. 見出しにタイムスタンプは入れない

台本の見出しに秒数表記（例: `(00:00~)`）は入れない。

## 品質チェックリスト

### フォーマットチェック

- [ ] タイトルが `# [台本タイトル]` 形式になっている
- [ ] 冒頭情報が「ジャンル」「想定時間」「作成日」のみになっている
- [ ] セクションが `## N. セクション名` 形式で連番になっている
- [ ] 区切り線 `---` が台本内に含まれていない
- [ ] 【スタート】が1. オープニングの冒頭にある
- [ ] 【終わり】が最終セクションの最終文冒頭にある

### 共通チェック項目

- [ ] 開始文言が「【スタート】こんにちは、Build AI Academyのたんのです」になっている
- [ ] 終了文言が「【終わり】それでは次の講座で会いましょう。ありがとうございました。」になっている
- [ ] 専門用語・横文字にすべて説明が付いている
- [ ] 全体が初心者にも理解しやすい表現になっている
- [ ] 住宅会社目線の内容が含まれている

### 概念系専用チェック項目

- [ ] 前提知識が結論より先に説明されている
- [ ] 専門用語に平易な説明が添えられている
- [ ] 事例が「セーフ」と「アウト」の両面で紹介されている
