# Google AI Studio 情報収集レポート

**調査日**：2025年12月18日

## 1. ツール概要

### ツール名
Google AI Studio（グーグル AI スタジオ）

### 開発元
Google（Alphabet Inc.）

### リリース時期
- 初期リリース：2023年末頃（Gemini API公開時）
- Build機能：2025年5月のGoogle I/Oで発表
- UI刷新：2025年4月

### 一言説明
Google AI Studioは、GoogleのGeminiモデルをブラウザ上で無料で試せるAI開発プラットフォームです。テキスト、画像、動画、音声など多様なデータを扱い、プロンプト作成からアプリ開発、APIキー取得まで一貫して行えます。

### 主要機能一覧
- **プロンプト作成・テスト**：テキスト入力でGeminiモデルと対話
- **Build（ビルド）機能**：自然言語でWebアプリを自動生成
- **Live機能**：マイク・カメラ・画面共有でリアルタイム対話
- **Generate Media**：画像生成（Nano Banana、Imagen）、動画生成（Veo）、音声生成
- **マルチモーダル入力**：テキスト、画像、動画、音声、PDF、YouTubeのURL対応
- **APIキー発行**：外部アプリ連携用のAPIキー取得
- **Cloud Runデプロイ**：作成したアプリをワンクリックでデプロイ
- **コンテキストキャッシング**：長文処理のコスト削減
- **Google検索連携（Grounding）**：最新情報を参照した回答生成

## 3. 制約・制限事項

### 無料プランの制限

#### レート制限（RPM: 1分あたり / RPD: 1日あたり）

| モデル | RPM | RPD |
|--------|-----|-----|
| Gemini 2.5 Pro | 5回/分 | 25回/日 |
| Gemini 2.5 Flash | 10回/分 | 500回/日 |
| Gemini 2.0 Flash | 15回/分 | 1,500回/日 |

#### その他の制限
- 動画生成（Veo 2）：1日3〜5回まで
- 画像生成：1回のプロンプトで4枚まで
- 動画：最大8秒まで
- TPM（1分あたりのトークン数）制限あり

### 無料枠のリセットタイミング
- 毎日、米国太平洋時間0時にリセット
- 日本時間では16時頃（夏時間）〜17時頃（標準時間）

### 有料プランの利用層（Usage Tiers）

| 層 | 条件 |
|---|---|
| Free | 対象国のユーザー |
| Tier 1 | 有料の課金アカウントが必要 |
| Tier 2 | 累計支出$250以上、支払い成功から30日以上経過 |
| Tier 3 | 累計支出$1,000以上、支払い成功から30日以上経過 |

### 対応環境

| 項目 | 対応状況 |
|------|---------|
| 対応ブラウザ | Chrome（推奨）、Edge、Firefox、Safari |
| モバイルアプリ | なし（ブラウザからアクセス） |
| 日本語対応 | プロンプトは日本語対応、UIは基本英語（ブラウザ翻訳で対応可能） |

### データ学習に関する重要事項
- **無料プラン**：入力データがGoogleの製品改善に使用される可能性あり
- **有料プラン**：コンテンツは製品改善に使用されない
- 「学習させない設定」は無料版では不可

## 5. 画面・操作関連情報

### アカウント作成・初期設定の手順

1. **公式サイトにアクセス**
   - https://aistudio.google.com にアクセス

2. **Googleアカウントでログイン**
   - 既存のGoogleアカウントでサインイン
   - なければ新規作成

3. **利用規約に同意**
   - プライバシーポリシーを確認し同意

4. **即座に利用開始**
   - ログイン後すぐにプロンプト入力可能

### 画面構成（2025年4月刷新後）

#### メインUI構成
- **左側：ナビゲーションメニュー**
  - プロジェクト一覧
  - モデルライブラリ
  - 設定画面へのショートカット

- **中央：ワークスペース**
  - プロンプト入力エリア
  - コード編集エリア
  - プレビュー表示

- **上部：アクションバー**
  - 一般的なタスクへのクイックアクセス

- **右側：出力表示エリア**
  - AIの生成結果
  - APIレスポンス
  - エラー情報

#### 主要タブ

| タブ名 | 機能 |
|--------|------|
| Dashboard | APIキー管理、使用量確認、変更ログ |
| Prompts | プロンプト作成・テスト |
| Build | Webアプリの自動生成 |
| Live | リアルタイム音声・映像対話 |
| Generate Media | 画像・動画・音声生成 |

### Build機能の操作フロー（詳細）

#### 基本フロー

1. **Buildタブを選択**
   - 左側メニューから「Build（実験的機能）」をクリック

2. **プロンプト入力**
   - 作りたいアプリの説明を自然言語で記述
   - 例：「住宅の間取り図をアップロードすると部屋の面積を計算するアプリ」

3. **機能カードの選択（オプション）**
   - プロンプト入力欄下部のカードから追加機能を選択
   - 15種類以上の機能カード：
     - 画像生成・編集・アニメーション
     - 音声アプリ・音声文字起こし
     - チャットボット
     - Google Maps統合
     - YouTube連携
     - 検索データ連携

4. **ビルドボタンをクリック**
   - AIが自動でコードを生成
   - 生成時間：数分〜数十分

5. **プレビューで確認**
   - 生成されたアプリをブラウザ上でテスト

6. **注釈機能で修正**（2025年10月追加）
   - プレビュー上に直接描画やコメントを追加
   - 自然言語で修正指示
   - 例：「ボタンを赤くして」「もっと大きく」

7. **デプロイ（オプション）**
   - 右上のロケットボタン → 「Deploy App」
   - Cloud Runにワンクリックでデプロイ

#### Build機能のデフォルト設定
- モデル：Gemini 2.5 Pro
- フレームワーク：React（TypeScript）またはAngular
- スタイリング：Tailwind CSS

#### Build機能の制限事項
- サーバー通信が必要な大規模アプリには不向き
- 複雑な条件分岐ではバグが発生する可能性あり
- 共有アプリへのアクセスにはGoogleアカウントが必要

## 7. 注意点

### 導入時の注意

1. **データプライバシーの理解**
   - 無料プランでは入力データが学習に使用される可能性
   - 機密情報・個人情報は入力しない
   - 商用で機密データを扱う場合は有料プラン推奨

2. **レート制限の把握**
   - 特にGemini 2.5 Proは1日25回の制限
   - 制限に達したら翌日（日本時間16〜17時）まで待機

3. **UIは基本英語**
   - 日本語UIの公式設定はなし
   - Chrome翻訳機能で対応可能

### 運用上の注意

1. **無料枠の管理**
   - 使用量はDashboardタブで確認可能
   - 意図せず有料利用にならないよう注意

2. **Cloud Runデプロイ時のコスト**
   - デプロイしたアプリへのアクセスが多いと課金発生
   - プロトタイプ・個人利用の範囲で使用推奨

3. **セキュリティリスク**
   - 間接プロンプトインジェクションの可能性
   - 機密情報を含むドキュメントは避ける

### よくあるつまずきポイント

1. **レート制限エラー**
   - 「429エラー」が出たら制限に達した証拠
   - 翌日のリセットを待つか、別モデルを使用

2. **Build機能でのエラー**
   - ファイル形式の混在（JPGとPNG）で問題発生することあり
   - 複雑なロジックは手動修正が必要な場合も

3. **日本語出力の精度**
   - モデルによって日本語の自然さに差がある
   - Gemini 2.5系は日本語対応が比較的良好

4. **動画生成の制限**
   - Veo 2は1日3〜5回まで
   - 8秒の短い動画のみ対応

## 10. Gemini 3.0（Gemini 3 Pro / Flash）詳細

### 概要
Gemini 3.0は2025年11月にGoogleがリリースした最新のAIモデルファミリーで、「最も知的なモデルファミリー」と位置づけられています。高度な推論能力と自律機能が大幅に向上しています。

### モデルの種類

| モデル | 特徴 | リリース |
|--------|------|----------|
| **Gemini 3 Pro** | 最高性能の推論・マルチモーダル理解 | 2025年11月 |
| **Gemini 3 Flash** | Proの4分の1以下のコストで高速処理 | 2025年12月17日 |

### Gemini 3 Pro の特徴
- **マルチモーダル対応**：テキスト、画像、動画、音声、PDFを入力可能
- **入力トークン**：最大100万トークン（1,048,576）
- **出力トークン**：最大65,536トークン
- **知識カットオフ**：2025年1月
- **対応機能**：
  - Function Calling（関数呼び出し）
  - Structured Outputs（構造化出力）
  - Code Execution（コード実行）
  - File Search（ファイル検索）
  - Thinking Features（思考機能）
  - Search Grounding（検索連携）
- **非対応機能**：画像生成、Live API

### Gemini 3 Flash の特徴
- **コンセプト**：「速度のために構築された最も知的なモデル」
- **性能**：多くのベンチマークでGemini 2.5 Proを上回る
- **博士レベルの推論**：
  - GPQA Diamond：90.4%
  - Humanity's Last Exam：33.7%（ツールなし）
- **2つのモード**：
  - **Fast**：素早い回答用
  - **Thinking**：複雑な問題用（深い推論）

### 思考レベル（Thinking Level）設定

| レベル | 用途 |
|--------|------|
| `minimal` | 最小限の推論（Flashのみ） |
| `low` | 低遅延・高スループット向け |
| `medium` | 中程度の推論（Flashのみ） |
| `high`（デフォルト） | より深い推論 |

### Gemini 3 の料金

| モデル | 入力（/100万トークン） | 出力（/100万トークン） |
|--------|----------------------|----------------------|
| Gemini 3 Pro | $2〜$4 | $12〜$18 |
| Gemini 3 Flash | $0.50（テキスト/画像/動画）/ $1.00（音声） | $3.00 |

### 無料枠の有無
- **Gemini 3 Flash**：無料枠あり（Google AI Studioで利用可能）
- **Gemini 3 Pro**：現時点ではAPIに無料枠なし（Google AI Studioでは試用可能）

### 利用可能なプラットフォーム
- Google AI Studio
- Geminiアプリ/Web版（gemini.google.com）
- Google検索のAIモード
- Vertex AI
- Antigravity（AIコード開発プラットフォーム）
- Android Studio
- Gemini CLI

### 重要な注意点
- **Temperature設定**：デフォルト値の1.0を維持することを強く推奨
- **思考署名（Thought Signatures）**：関数呼び出しや画像生成では厳密に検証される
- **対応ツール**：Google Search、File Search、Code Execution、URL Context
- **未対応**：Maps Grounding、Computer Use

### Google AI StudioでのGemini 3の使い方（詳細手順）

#### 無料で使えるか？
**はい、無料で使えます。** Google AI Studioでは、Gemini 3 Pro/Flashをモデルとして選択し、無料で試すことができます。ただし、以下の制限があります：
- レート制限（1分あたり/1日あたりのリクエスト数）
- 入力データがGoogleの製品改善に使用される可能性
- API経由での利用には別途課金が発生

#### 具体的な使用手順

1. **Google AI Studioにアクセス**
   - https://aistudio.google.com にアクセス
   - Googleアカウントでログイン

2. **モデルを選択**
   - 右側のサイドバーで「Model」を確認
   - 「**Gemini 3 Pro Preview**」または「**Gemini 3 Flash**」を選択

3. **Thinking Level（思考レベル）を設定**
   - 深い推論が必要な場合は「**High**」を選択
   - 高速な回答が必要な場合は「Low」または「Fast」

4. **プロンプトを入力**
   - 画面下部のメッセージボックスにテキストを入力
   - ファイル（画像、PDF、動画など）も添付可能

5. **「Run」ボタンをクリック**
   - AIが回答を生成
   - 設定はそのままで使用可能（英語UIのまま日本語で質問OK）

#### APIキー警告への対応
- 使用時に「You have no Paid Project」という警告が出ても「Cancel」で閉じてOK
- テスト利用や記事作成程度なら無料枠で十分動作

#### 無料枠の目安
- Gemini 3 Flashは無料枠あり
- Gemini 3 ProはGoogle AI Studioで試用可能（APIには無料枠なし）
- 本1冊分程度のコンテンツまで処理可能

## 11. Nano Banana Pro（Gemini 3 Pro Image）詳細

### 概要
Nano Banana Pro（正式名称：Gemini 3 Pro Image）は、2025年11月20日にGoogleが発表した最新の画像生成AIモデルです。「推論する画像生成AI」という革新的なアプローチで、Gemini 3 Proの言語理解能力を画像生成に統合しています。

### モデルの種類

| モデル | 愛称 | 特徴 |
|--------|------|------|
| Gemini 3 Pro Image Preview | **Nano Banana Pro** | 高度な画像生成、最大4K解像度 |
| Gemini 2.5 Flash Image | **Nano Banana** | 高速・低遅延処理、1024px解像度 |

### Nano Banana Pro の主な進化点

1. **正確な文字・情報描写**
   - 複雑なテキストを含む画像の生成が可能
   - サムネイルやWebデザインで一文字の狂いもなく表示
   - **日本語対応が優秀**：ひらがな・カタカナ・漢字の混在、縦書き対応

2. **インフォグラフィック生成**
   - 数値データを数学的に理解
   - 正確な円グラフや棒グラフを自動生成
   - 売上比率やデータ可視化が自動化

3. **構図・照明の精密制御**
   - 「ローアングルから見上げた構図で逆光」などの専門的指示を理解
   - カメラワークレベルでの制御が可能

4. **最大4K解像度**

#### Geminiアプリでの使い方（無料）

1. **Gemini（gemini.google.com）にアクセス**

2. **「画像を作成」を選択**
   - または直接プロンプトで「〇〇の画像を作成して」と指示

3. **思考モード（Thinking Mode）をON**
   - 右上の設定から有効化
   - これでNano Banana Proが使用される

4. **プロンプトを入力**
   - 生成したい画像の説明を入力
   - 参照画像をアップロードすることも可能

5. **生成実行**
   - 数秒〜数十秒で画像が生成

#### Google AI Studioでの使い方（有料）

1. **Google AI Studioにアクセス**
   - https://aistudio.google.com にアクセス

2. **APIキーを作成**
   - 「Get API Key」をクリック
   - キー名を指定してプロジェクトを作成
   - **お支払い情報を設定**（請求先アカウントをリンク）

3. **Playgroundを開く**
   - モデルで「**Nano Banana Pro**」または「**nano-banana**」を選択

4. **APIキーを選択**
   - メッセージボックスの鍵アイコンをクリック
   - 作成したAPIキーを選択

5. **オプションを設定**
   - **Temperature**（0〜1）：創造性の度合い
   - **Aspect ratio**：1:1、9:16、16:9、3:4、4:3など
   - **Resolution**：1K / 2K / 4K
   - **Google Search連携**：ON/OFF

6. **プロンプトを入力して「Run」**
   - 「＋」ボタンで参照画像を追加可能
   - 最大14枚まで参照画像を使用可能

## 12. まとめ

Google AI Studioは、Googleが提供する無料のAI開発プラットフォームで、特に2025年5月に追加された**Build機能**が大きな特徴です。プログラミング知識がなくても、自然言語でWebアプリを自動生成でき、ワンクリックでCloud Runにデプロイできます。

**特に優れている点**：
- 無料で使える範囲が広い（競合より充実）
- Googleアカウントだけで即開始可能
- Build機能で素早くプロトタイプ作成
- マルチモーダル対応（テキスト、画像、動画、音声、PDF、YouTube）

**注意すべき点**：
- 無料プランでは入力データが学習に使用される可能性
- レート制限があり、特にGemini 2.5 Proは1日25回まで
- 機密情報や個人情報は入力しない

住宅業界での活用としては、物件資料の分析・要約、簡易的な顧客向けチャットボットの作成、プレゼン資料の下書き作成などが考えられます。
