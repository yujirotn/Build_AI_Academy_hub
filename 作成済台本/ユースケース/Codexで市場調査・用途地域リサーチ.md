# Codexで市場調査・用途地域リサーチ

ジャンル: ユースケース系
想定時間: 12分
作成日: 2026年3月13日

## 1. オープニング

【スタート】こんにちは、Build AI Academyのたんのです。

今回のテーマは「面倒なデータ調査を、AIにまるごとお任せする方法」です。

皆さん、こんな経験ありませんか？

営業会議の前日、「最近の住宅着工ってどうなってる？数字で見せて」と上司に言われて、国土交通省のサイトを開いて、PDFの統計資料を探して、数字を拾ってExcelに打ち込んで、グラフを作って……。気づけば2時間経っていた、なんてこと。

あるいは、お客さまから「この土地って何が建てられるの？」と聞かれて、市区町村の都市計画情報を調べに行くけど、サイトがわかりづらくて目的の情報にたどり着けない。結局、役所に電話して確認する、ということもあるかもしれません。

今日は、こういった「調べてまとめる」作業を、OpenAIのCodex（コーデックス）というアプリにお任せする方法をお見せします。

## 2. 従来の課題

こういった市場調査や土地情報のリサーチ、何が大変かというと、「必要な情報があちこちに散らばっている」ことなんですよね。

住宅着工統計ひとつとっても、国土交通省のサイト、都道府県の統計ページ、民間の調査レポートなど、データの出どころがバラバラです。しかもPDFで公開されていることが多いので、数字をコピーして貼り付けるだけでもひと苦労です。

用途地域の調査も同じです。自治体によって都市計画情報の公開の仕方が違いますし、建ぺい率や容積率、高さ制限などを調べるにはいくつものページを行ったり来たりしなければいけません。

こういった課題の本質は、「情報を集めて整理する」という単純だけど手間のかかる作業に時間を取られてしまっている、ということです。考えたり判断したりする時間ではなく、データを探す時間に追われてしまう。もったいないですよね。

## 3. 活用メリット

Codexを使うと、この「調べて→集めて→まとめる」という一連の流れをAIに任せられます。

**メリット1：調査からレポートまで一気通貫**。ChatGPTのチャットでは「まずこれを調べて」「次にこれをまとめて」と何度もやりとりが必要でしたが、Codexなら「○○を調べてレポートにして」と一回お願いするだけで、AIが自動で情報収集から整理、レポート作成まで進めてくれます。

**メリット2：データの集計やグラフ化も自動**。Codexは裏でプログラムを動かして、数値データをグラフにしたり、表に整理したりしてくれます。Excelで手作業でグラフを作っていた作業から解放されます。プログラミングの知識は一切いりません。AIが全部やってくれます。

**メリット3：調査のひな型ができる**。一度Codexに依頼した内容は、次回以降も同じような形で使い回せます。「毎月の着工統計チェック」のような定期的な調査なら、前回のお願い文をコピーして日付だけ変えれば、同じ品質のレポートがすぐに出来上がります。

## 4. 本題への導入

では、実際にCodexをどう使うか説明します。

Codexは、ChatGPTのアプリとは別に用意された「エージェント型」のツールです。エージェントというのは、指示を出したらAIが自分で考えて、調べて、作業を進めてくれる仕組みのことです。

使うには、ChatGPT Plus（月額約3,000円）以上のプランが必要です。Plus以上のプランに入っていれば、追加料金なしでCodexが使えます。期間限定で無料プランでもCodexを試せるキャンペーンが行われていることもありますので、公式サイトを確認してみてください。

使い方は3通りあります。
- **Codexアプリ**（macOSまたはWindows用のデスクトップアプリ）
- **Web版**（ブラウザからchatgpt.com/codexにアクセス）
- **CLI**（コマンドライン。こちらはプログラマー向けなので、今回は触れません）

今回はCodexアプリを使った実演をお見せします。

注意点として、Plusプランでは1日に実行できるタスク数がおよそ25件程度、同時に実行できるタスクは1つだけです。業務で本格的に使う場合は、1日のタスク配分を考えておくとスムーズです。

※料金やタスク数の制限は2026年3月時点の情報です。最新情報は公式サイトでご確認ください。

## 5. 実演パート

### ユースケース1：住宅着工統計データの収集・分析・グラフ化

Codexアプリを開いて、以下のようにお願いします。

```
2024年度と2025年度の住宅着工統計を調べて、
以下の内容をレポートにまとめてください。

調べてほしいこと：
・全国の新設住宅着工戸数（月別推移と年間合計）
・利用関係別の内訳（注文住宅・分譲住宅・賃貸住宅）
・前年同月比の推移
・地方部（三大都市圏以外）の着工動向

出力形式：
・主要な数値は表にまとめる
・月別推移はグラフで表示する
・最後に「住宅会社として押さえておくべきポイント」を
  3つにまとめる
```

Codexがこのお願いを受け取ると、自動でWebを検索し、国土交通省などの公開データを収集して、数値を整理し、グラフ付きのレポートを生成してくれます。

完成したレポートには以下のような内容が含まれます。
- 月別の着工戸数を並べた表
- 前年比の推移グラフ
- 注文住宅・分譲・賃貸の構成比
- 住宅会社視点でのまとめコメント

**ポイント**：お願い文に「出力形式」を具体的に書いておくと、自分が求めている形に近いレポートが出てきます。

### ユースケース2：用途地域データのリサーチ・整理

続いて、土地のリサーチです。こんなお願いを出してみます。

```
以下の住所の土地について、建築に関わる基本情報を
調べてまとめてください。

住所：東京都世田谷区○○○ X丁目Y番Z号

調べてほしいこと：
・用途地域
・建ぺい率・容積率
・高さ制限
・防火地域の指定
・前面道路の幅員（わかる範囲で）

出力形式：
・箇条書きで簡潔に
・各項目に情報の出典（どのサイトで確認したか）を付記
・最後に「この土地で注意すべきポイント」をまとめる
```

Codexが自治体の都市計画情報サイトなどを検索し、該当エリアの規制情報を集めてくれます。

**大事な注意点**：用途地域や建ぺい率などの法規制情報は、必ず正式な資料（自治体の窓口や都市計画図）で最終確認してください。AIの調査結果はあくまで「下調べ」です。お客さまへの説明や正式な書類に使う前には、必ず裏取りをしましょう。

## 6. クロージング

今日は、Codexを使って住宅着工統計の調査と用途地域のリサーチを自動化する方法をお見せしました。

この方法は、住宅着工統計や用途地域だけでなく、たとえば「地域の人口動態を調べて営業エリアの将来性を分析する」「建材メーカーの新製品情報を集めて比較表にする」「補助金や助成金の最新情報をまとめる」など、あらゆる調査業務に応用できます。

「まずはCodexにざっくり調べてもらって、その結果を自分で確認・修正する」という使い方を習慣にすると、調査にかける時間がぐっと楽になります。完璧な調査を一発で出してもらう必要はありません。「ゼロから調べる」のと「AIが集めてくれた情報を確認する」のでは、負担がまったく違います。

皆さんの営業エリアに合わせて、住所や調べたい項目を変えて試してみてください。たとえば、自社の商圏エリアの着工統計だけに絞って調べるとか、よくお客さまから聞かれる地域の用途地域情報をまとめておくとか。一度やり方を覚えれば、繰り返し使えます。
